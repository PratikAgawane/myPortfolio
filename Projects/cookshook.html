Overview

CookShook is a smart recipe recommendation system built using Flask, Machine Learning, and a structured food dataset. Users can search recipes, save favorites, plan meals, and receive personalized recommendations based on their tastes.

Features

  • User login & registration
  • Personalized recipe recommendations
  • Meal planner
  • Favorites and rating system
  • Admin dashboard
  • Email-based OTP verification (Forgot Password)

Tech Stack

  • Python, Flask
  • HTML, CSS, JavaScript, Bootstrap
  • Machine Learning (recipe similarity)
  • MySQL Database
